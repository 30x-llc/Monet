## Cómo se llena este archivo

Para cada plantilla, Juan Diego me pasa:
1. Link del archivo de Figma.
2. Frame específico que es la "plantilla base" (la que duplico).
3. Qué elementos se reemplazan típicamente (logo cliente, fondo, copy, nombres).
4. Convenciones de naming dentro del archivo (capas, frames, components).

Yo agrego: file key (extraído del link), fecha de última actualización vista, notas operativas.
