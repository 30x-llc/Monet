Archivos preservados de iteraciones anteriores de Monet.

## `monet-app/`

App Next.js completa que vivía en `monet.30x.com`. Stack: Next.js 16 + React 19 + Tailwind v4 + Neon Postgres + Vercel Blob + Anthropic Claude + Perplexity/Exa + Canva Connect.

Generaba propuestas comerciales desde briefing corto, con un canvas estilo Figma para edición, exportable a PDF y a Canva. Buen prototipo. La razón por la que se reemplazó: la generación pura no llegaba al estándar de Juan Diego, y vivir en una web app aparte rompía el loop del equipo (que trabaja en Slack).

Decisiones operativas pendientes (Juan Diego):

- ¿Qué pasa con el subdominio `monet.30x.com` y el alias `design.30x.com`?
- ¿Se reusan piezas en el futuro (ej. una UI web para el agente)?

Mientras se deciden, la app queda acá congelada como referencia.
