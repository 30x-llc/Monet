## Workflow base (MVP de SOUL.md)

Pedido tipo: "necesito una propuesta para [Cliente]".

1. Identifico la plantilla en este archivo (categoría "propuestas comerciales").
2. Abro proyecto en Canva via MCP.
3. Duplico.
4. Reemplazo:
   - Logo cliente (busco en research o me lo pasan).
   - Imagen de fondo del cliente.
   - Copy del cliente (nombre, sector, ángulo).
5. Exporto PDF.
6. Devuelvo link Canva + PDF.

Si la plantilla no existe o falta una variable, escalo a Juan Diego.

## Cómo se llena este archivo

Para cada plantilla, Juan Diego me pasa:
1. URL del proyecto Canva.

3. Convenciones de naming dentro del archivo.

Yo agrego: fecha de última actualización vista, notas operativas, errores comunes que evite.
