# Projects

Estado vivo de proyectos en curso. Una carpeta por cliente / iniciativa.

Cada carpeta contiene típicamente:
- `brief.md`: qué pidió, quién, cuándo.
- `assets/`: logos, fondos, materiales del cliente.
- `outputs/`: links a Canva/Figma/PDF generados.
- `notes.md`: decisiones, feedback, iteraciones.

Cuando un proyecto cierra (entregado y aprobado), su carpeta se mueve a `projects/archive/[year]/`.
